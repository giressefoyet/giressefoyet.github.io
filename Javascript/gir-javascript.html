Cours JavaScript

Leçon 1:

Ce cours permet d'obtenir les bases dans le langage de balisage HTML, et plus précisément le HTML5, qui a été conçu pour l'insertion du contenu des pages web. Pour être d'une originalité débordante, nous allons parcourir les différents chapitres de ce cours étapes par étapes tout en créant d'un site web volontairement simple.

CSS

Le CSS est un langage à part entière qui permet de mettre en forme les balises HTML. C'est grâce à lui que nous allons pouvoir définir notamment les couleurs de fond, la taille des textes et la présentation globale d'un site. Nous poursuivrons la mise en forme de notre site web d'exemple en utilisant la dernière version, le CSS3.

Javascript

Le JavaScript (abrégé en JS) est un langage de développement principalement utilisé pour les sites et applications web. Les navigateurs web disposent d'un moteur JavaScript dédié pour l'interpréter. Avec le HTML et le CSS, le JS fait partie des technologies incontournables du web. Le JS ne doit pas être confondu avec le langage de programmation Java.

Php

Le PHP est un langage de programmation web qui s'exécute côté serveur. Il permet notamment de rendre un site web dynamique. Très couramment utilisé, il permet de répondre à de nombreux besoins. Le PHP se cache derrière bon nombre de sites. En 2018, 83.4 % des serveurs web l'utilisent.

MySql

Le cours sur MySQL permet d'apprendre le langage SQL et de découvrir MySQL. C'est un Systèmes de Gestion de Bases de Données (SGBD) qui permet d'intéragir avec les informations d'une base de données. On entend par intéragir, sélectionner, ajouter, modifier et supprimer des données de la base

WordPress

Dans cette formation jnous allons découvrir le CMS WordPress, son installation et son utilisations. Nous verrons également les plugins, les éditeurs de contenus, les thèmes, créatioons des menus - page d'acceuil - widgets - gestions des médias , etc. ceux-ci autour d'un site web simpa à réaliser.

Infographie

Les objectifs de cours seront de Maîtriser les notions de base de l’infographie, Savoir générer, modifier et manipuler des images, Savoir générer, modifier et manipuler des modèles 3D, Comprendre les notions de géométrie (2D et 3D) sous-jacentes, Connaître certaines structures de données utilisées en infographie, Connaître certains algorithmes utilisés en infographie, Savoir les implémenter, Développer des applications graphiques en 3D, Communiquer à l’oral et à l’écrit ces différents aspects...
